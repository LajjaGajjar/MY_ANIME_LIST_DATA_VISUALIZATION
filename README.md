# 📌 Project Overview

The **MyAnimeList Anime Data Analysis Dashboard** is an interactive Power BI project that analyzes the **Top 10,000 anime titles** available on MyAnimeList.

The objective of this project is to transform raw anime data into meaningful visual insights by exploring:

- Anime ratings
- Popularity
- Episode counts
- Anime types
- Release trends
- Audience engagement

The dashboard allows users to interactively filter and explore anime statistics using Power BI slicers and dynamic visuals.

---

# 🎯 Objectives

- Analyze anime ratings and rankings.
- Identify the highest-rated anime.
- Study anime production trends over time.
- Compare different anime types.
- Analyze audience engagement through member counts.
- Identify long-running anime series.
- Create an interactive dashboard using Power BI.

# 📂 Dataset

**Dataset Name**

Top 10,000 Anime Dataset

**Source**

MyAnimeList (MAL)

The dataset contains information about approximately **10,000 anime** and follows a **3rd Normal Form (3NF)** relational structure.

### Main Attributes

| Column | Description |
|---------|-------------|
| anime_id | Unique Anime ID |
| title | Anime Title |
| score | Average MyAnimeList Rating |
| rank | Overall Rank |
| popularity | Popularity Ranking |
| members | Number of MAL Members |
| synopsis | Story Description |
| start_date | Release Date |
| end_date | End Date |
| type | TV, Movie, OVA, ONA, Special |
| episodes | Number of Episodes |
| image_url | Anime Poster |

# 🔍 Key Insights

## 📺 TV Anime Dominate

TV anime account for the largest proportion of titles in the dataset, indicating that television remains the primary production format.

---

## ⭐ Highly Rated Anime Are Rare

Only a limited number of anime achieve ratings above **9.0**, highlighting the competitive nature of MyAnimeList ratings.

---

## 📈 Anime Production Has Increased

Anime production has grown steadily over the years, with a noticeable increase after the early 2000s, reflecting the industry's global expansion.

---

## 🎬 Long-Running Anime Are Uncommon

Most anime consist of seasonal releases with relatively few episodes, while only a handful of titles exceed several hundred episodes.

---

## 👥 Strong Audience Engagement

The dataset represents approximately **1 billion cumulative MyAnimeList member entries**, demonstrating the platform's extensive user community.
